# Jarvis AI-Bridge

Canal isolado de comunicacao para tarefas e revisoes do projeto `jarvis-assistant`.

Estrutura:

- `to-executor/`: instrucoes para o executor agir no projeto Jarvis
- `state/`: estado atual do ciclo deste subprojeto

Este bridge existe para nao misturar as demandas do Jarvis com o bridge principal do repositiorio.
